TYPOGRAPHY

FAN RESCUE — DISPLAY TYPE
Archivo Black, 900 weight. Used for headings, the wordmark, and the FR monogram. Letter-spacing 0.04em–0.12em depending on use. Loaded automatically from Google Fonts when you include tokens.css.
var(--fr-font-display) — 'Archivo Black'
Source Sans 3 — Body Type
Used for all body copy, the tagline, UI labels, and supporting text. Available weights 300, 400, 500, 600, 700. Loaded automatically from Google Fonts. The full ESTABLISHED 2005 tagline uses this at 0.25em letter-spacing.
var(--fr-font-body) — 'Source Sans 3'
